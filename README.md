# 🟩 Wordly — игра в слова на Python

Угадай слово из 5 букв за 6 попыток.

- 🟩 **Зелёный** — буква на своём месте
- 🟨 **Жёлтый** — буква есть в слове, но не на месте
- ⬜ **Серый** — буквы нет в слове

---

---

## 📂 Структура проекта

- `gui_wordly.py` — оконная версия (CustomTkinter)
- `console_wordly.py` — версия для терминала
- `game.py` — игровая логика
- `words.py` — загрузка слов
- `n5.txt` — словарь (5-буквенные слова)
